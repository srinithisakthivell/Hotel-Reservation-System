# 🏨 Hotel Reservation System

A Java-based console application that simulates a hotel booking system using Object-Oriented Programming principles.

---

## 🔧 Features

- Room booking and cancellation  
- Multiple room types (Single, Double, Suite, Deluxe, Presidential Suite)  
- Reward points system for customers  
- Customer rating system  
- Booking history tracking  
- Menu-driven user interface  

---

## 💡 Concepts Used

- Interfaces and Abstract Classes  
- Inheritance and Polymorphism  
- Encapsulation and Modular Design  
- Collections (ArrayList, HashMap)  
- Basic exception-safe input handling

## ▶️ How to Run

1. Open terminal and navigate to project folder:

cd src

2. Compile the program:

javac main/Main.java

3. Run the application:

java main.Main
